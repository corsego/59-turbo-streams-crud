### Setup:

* git clone git@github.com:secretpray/59-turbo-streams-crud.git

* cd 59-turbo-streams-crud

* git checkout turbo_frame

* bundle

* yarn install --check-files

* ./bin/rails db:setup

* ./bin/rails s
